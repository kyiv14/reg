# 🚀 SEO Beauty Parser v1.0

Профессиональный PHP-инструмент для автоматизированного сбора базы доменов в бьюти-нише (салоны красоты, SPA, косметология).

## 🛠 Технологии
- **Backend:** PHP 8.x + cURL (с имитацией сессий и Cookies).
- **Frontend:** HTML5, CSS3, JavaScript (AJAX/Fetch API).
- **Security:** Исключение дубликатов через хеш-таблицы (array_flip).

## 📋 Установка
1. Склонируйте репозиторий в папку на сервере (например, `/reg/`).
2. Установите права доступа `777` на папку проекта.
3. Откройте `parser.php` в браузере.

## ⚠️ Отладка
Если сбор не начинается, проверьте файл `error_log.txt`. Основная причина — блокировка IP сервера со стороны реестра (Connection Timeout).
