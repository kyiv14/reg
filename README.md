# 🛡️ Universal Domain Parser (PHP/Python)

Профессиональный инструментарий для анализа доменных зон и сбора коммерческих данных в нише Beauty. 

### Структура:
- **PHP Version**: Интерактивный веб-интерфейс для работы на хостинге i-tv.top.
- **Python Version**: Асинхронный движок для высокоскоростного сбора данных.

### Как использовать:
1. Для веба: загрузите содержимое `php_version` на сервер.
2. Для консоли: `pip install -r requirements.txt && python python_version/parser.py`
